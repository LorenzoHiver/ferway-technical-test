# Installer l'application en local

Version de node : 18.7.0. 

Version de yarn : 1.22.19. 


1 - Clone le projet. 

2 - Executer un yarn à la racine. 

3 - Lancer la commander yarn start. 

4 - Accéder en localhost sur le port 3000.
